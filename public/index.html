DevCollab

It is a full-featured social collaboration backend application designed with a focus on modularity, scalability, and clean REST API architecture. It includes:

  • Authentication and authorization with JWTs
  • User profiles with customizable avatars, skills, and social links
  • Post creation with media uploads and feed management
  • Comments with CRUD and like/unlike functionality
  • Follow and unfollow users with followers/following lists
  • Real-time chat messaging between users using WebSocket
